# Skills

个人整理的 Agent Skills，可与 Cursor / Claude / Windsurf 等 AI IDE 配合使用。

## 技能列表

| 技能 | 说明 |
|------|------|
| **resume-content-optimization** | 简历内容优化：去夸大表述、JD 对齐、结构梳理、事实核查 |
| **resume-html-layout** | 将 Markdown 简历转为双页 HTML 排版（侧栏+主栏、打印友好、精确分页） |

## 使用方式

将对应技能目录（含 `SKILL.md`）放到你的项目 `.agent/skills/` 或 IDE 配置的技能路径下即可被识别。

## 来源

上述技能由简历优化与 MD→HTML 排版实践提炼，详见 [health-intelligence](https://github.com/tiandee/health-intelligence) 项目。
